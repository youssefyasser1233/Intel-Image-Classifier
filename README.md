# Intel Image Classifier 🧠🖼️

This project is a deep learning-based image classification system that classifies images into different Intel-related categories using a CNN model.

---

## 🚀 Features
- CNN-based image classification model (PyTorch)
- Web interface for uploading images
- Real-time prediction results
- Trained on Intel Image Dataset

---

## 🧠 Model
The model is a Convolutional Neural Network trained to recognize multiple image classes.

- Input: Images
- Output: Predicted class label
- Framework: PyTorch
